# KeskOS Kvantum Theme

This optional Kvantum theme is staged for KeskOS systems that install the Kvantum engine.

It is derived from a dark Kvantum SVG theme structure and recolored to match the KeskOS palette:

- `#050505`
- `#080706`
- `#11100e`
- `#ce6a35`
- `#b8afa6`

KeskOS does not depend on Kvantum for the default System Settings experience.
If Kvantum is absent, the theme stack falls back to Breeze plus the `KeskOSDark` KDE color scheme.
