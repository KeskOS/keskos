# Old Wolfi / Rofi Launcher Archive

This directory contains the retired KeskOS launcher stack that was replaced by the
patched Plasma `Kesk Kickoff` launcher.

Archived here:

- `configs/rofi/`
- `configs/plasmoids/com.keskos.launcherbutton/`
- `configs/plasmoids/org.keskos.launcher/`
- `launcher/kesk_runner/`
- retired launcher desktop entries
- retired helper scripts and backend files
- old launcher docs

The active ISO no longer installs these components by default.

They were moved here so the old implementation can still be inspected or restored
later if needed, without keeping it wired into the current build.
